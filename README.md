# Leurs247 Brew-tap

## How do I install these formulae?

`brew install leurs247/brew-tap/<formula>`

Or `brew tap leurs247/brew-tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "leurs247/brew-tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
